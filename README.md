## 加入服务器：
- 使用版本1.19.2~1.7版本之间  **建议1.19.2**。
- 前往官方皮肤站[注册账户](https://skin.tesf.top/)。
- 登录到客户端，开始游戏。
- 点击多人游戏》添加新服务器》服务器名称`Galaxy Cube`，填写IP`mc.tesf.top`。
